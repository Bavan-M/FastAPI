What are we building?
A production-ready Gen AI chat application backend that combines every concept from Phase 5:
User opens browser
    ↓
WebSocket connection (bidirectional)
    ↓
User sends message
    ↓
Server calls LLM (with timeout + retry + circuit breaker)
    ↓
LLM response streams token by token back to user
    ↓
SSE broadcasts typing indicators to other users in same room
    ↓
Chat history stored per session
    ↓
Multiple rooms, auth, connection management
