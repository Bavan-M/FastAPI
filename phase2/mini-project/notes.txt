What are we building?
A fully functional Task Manager API where:

Users can register and login
Each user manages their own tasks (CRUD)
Admins can see all users and all tasks
Every route is protected with token-based auth
Clean error handling throughout
Proper project structure with routers, schemas, dependencies
Lifespan manages app resources
